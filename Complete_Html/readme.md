
#  HTML Practice Project

This project is a basic HTML webpage created for learning and practicing web development concepts.

---

##  Features

-  Simple Login Form (Username & Password)
-  Checkbox Inputs (Google, Facebook)
-  Image Links (GitHub Profile, Discord, ChatGPT)
-  Informational Paragraph about Nature
-  Basic Table (Student Data)
-  Dropdown Menu (Provinces of Pakistan)
-  Embedded YouTube Video
-  Feedback Textarea
-  Structured Layout (Header, Main, Footer)

---

##  Technologies Used

- HTML5
